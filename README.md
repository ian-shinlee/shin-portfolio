## 🛠 Tech Stack

* HTML5 / CSS3
* SASS / Tailwind CSS / Bootstrap
* JavaScript (ES6+) / jQuery
* React
* Git / GitHub / NPM

---

## ⚙️ Getting Started

### 1. 설치

```
npm install
```

### 2. 개발 서버 실행

```
npm start
```

👉 http://localhost:3000 에서 확인 가능

---

## 📦 Build

```
npm run build
```

프로덕션용으로 최적화된 파일이 생성됩니다.
